# Framework(pytorch)
효율성을 위한 지극히 개인적인 실험 Framework.
앞으로 유용한 tool 이나 function들을 추가 예정.
실험은 `./run_experiments.sh` 로 실행 가능.

## configs.py
실험의 기본 세팅을 알아보기 편하게 하기 위해 arg_parser를 별도로 구성함.
실험 상황에 맞게 편집 가능.
